## 一、设计意图：为什么加入轻战斗

### 原有问题

v1.1/v2.0 的交锋感全部集中在 QTE 阶段，玩家在此之前几乎没有主动行为——探索是被动的（等线索），接触是被动的（走过去），连接近异兽都只是"小心不要被发现"。这导致每局缺乏主动的操作高潮。

### 加入战斗的目标

- **让接近异兽的过程本身有操作感**：不只是"绕后走"，而是"判断攻击时机、主动出手"
- **让地形更有意义**：竹林可以躲、瘴气会掉血——玩家开始真正阅读地图
- **让资源决策更紧张**：血量是有限资源，带恢复道具就要少带封灵器
- **让异兽更有个性**：被动/领地/主动/伏击四种行为类型，让不同异兽有不同的"对局记忆"

### 轻战斗的边界

这不是 ARPG。战斗是**辅助捕获**的手段，不是目的本身。玩家无法靠战斗"杀死"异兽，只能削弱、控制、创造背刺窗口。最终捕获仍需通过 QTE + 封灵器完成。

## 二、生命值系统

### 2.1 基本设定

| 参数 | 数值 |
|------|------|
| 初始生命值 | **10 滴**（水墨滴图标，满格朱砂色） |
| 单局内上限 | 10 滴（不可超越上限） |
| 跨局恢复 | **不恢复**（HP 跨局持久化，不会因探索结束自动回满） |
| 溃散恢复 | 灵气溃散（死亡）后恢复至 **1 滴** |
| 局内恢复途径 | 仅通过消耗品（见§七） |
| 局外恢复途径 | 仅通过消耗品（PrepareScreen 可使用灵泉丸/绛珠露恢复） |
| 生命归零后果 | 触发**灵气溃散**状态（见§2.3） |

> **设计核心**：血量跨局持久化，不会自动恢复——这意味着多次探索的累积消耗是真实压力。玩家必须在准备阶段决定：是否用恢复道具补满血再出发，还是冒着残血的风险节省资源。溃散后保底恢复 1 滴，确保玩家永远有"最后一搏"的机会。

### 2.2 伤害来源

| 来源 | 伤害值 | 备注 |
|------|--------|------|
| 瘴气地形 | 1滴 / 4秒 | 持续计时，离开瘴气立即停止 |
| 普通攻击（异兽） | 1滴 | 大多数异兽的基础攻击 |
| 强力攻击（异兽） | 2滴 | 特定异兽的重击、范围攻击 |
| 噬天吞噬（特殊） | 3滴 | 极罕见，噬天的 SSR 专属技 |
| 压制失败反击 | 1滴 | 部分领地型异兽在 QTE 失败后反击一次 |

### 2.3 灵气溃散（生命归零）

当生命值降至 0，不是立即结算，而是进入**灵气溃散**状态：

```
HP = 0
  ↓
【灵气溃散状态·持续 8 秒】
  · 玩家移速 × 0.5（虚弱）
  · 视野半径收缩至 2.5 格
  · 所有已捕获灵契中，SSR 立即不稳定（须撤离时 QTE）
  · 场上所有异兽感知到玩家位置（无论距离）
  ↓
8 秒后 → 强制触发【紧急逃脱】
  · 随机丢失 1 只已捕获灵契（按 SSR→SR→R 顺序）
  · 灵石损失 40%
  · 兽魂/天晶清零
  · 传送至最近撤离点，直接进入结算

例外：若携带【复苏符】，溃散瞬间自动消耗，回复 4 滴血，跳过溃散流程。
```

> **设计意图**：灵气溃散给玩家 8 秒的"死前挣扎"时间，制造紧张感而非瞬死挫败感。玩家在这段时间内可以尝试跑向撤离点，但视野和速度都受限，几乎不可能再捕获异兽。

### 2.4 血量显示

- **UI 位置**：屏幕左上角，10 个水墨滴排列（横向）
- **满格**：朱砂红色实心滴
- **半格（未使用）**：无半格，整滴扣除
- **溃散预警**：生命值 ≤ 3 时，血条开始缓慢闪烁
- **受击反馈**：屏幕边缘短暂泛红（0.3 秒），水墨滴变灰并落下动画

## 三、异兽战斗行为分类

### 3.1 四种行为类型

所有异兽在原有 AI 状态机基础上，新增**战斗行为类型**标签：

| 类型 | 名称 | 核心特征 | 代表异兽 |
|------|------|---------|---------|
| 🕊 被动型 | Passive | 永不主动攻击，只会逃跑 | 白泽、冰蚕 |
| 🗿 领地型 | Territorial | 在固定范围内巡逻，被侵入则警告→攻击 | 石灵、水蛟、土偶 |
| 🔥 主动型 | Aggressive | 感知到玩家后主动追击并攻击，不轻易逃跑 | 噬天、雷翼、墨鸦 |
| 👁 伏击型 | Ambush | 隐匿状态游荡，玩家走近时突然现身并攻击 | 风鸣、玄狐（SSR） |

### 3.2 战斗行为对 AI 状态机的扩展

原有状态机在 `alert` 之后新增分支：

```
                    ┌─────────┐
                    │  idle   │
                    └────┬────┘
                         │
                    ┌────▼────┐
                    │ wander  │
                    └────┬────┘
                         │ 感知到玩家
                    ┌────▼────┐
                    │  alert  │ ← 所有类型共有
                    └────┬────┘
           ┌─────────────┼──────────────┐
           │             │              │
    [被动型]│      [领地型]│       [主动/伏击型]
           ▼             ▼              ▼
      ┌─────────┐  ┌──────────┐  ┌──────────────┐
      │  flee   │  │  warn    │  │    chase     │
      │ 逃离玩家 │  │ 警告停顿  │  │ 主动追击玩家  │
      └─────────┘  └────┬─────┘  └──────┬───────┘
                        │ 3秒内           │ 进入攻击距离
                        │ 玩家未撤离      │
                        ▼                ▼
                   ┌─────────┐    ┌──────────────┐
                   │ attack  │    │    attack    │
                   │ 发动攻击 │    │  发动攻击     │
                   └─────────┘    └──────────────┘
```

### 3.3 各类型详细行为规则

**🕊 被动型**
- `alert` 后直接进入 `flee`，不攻击
- 若玩家追赶超过 10 秒仍未甩掉，随机逃向地图边缘并消失（该只异兽本局消失，但不计入逃跑损失）

**🗿 领地型**
- 有一个固定"领地半径"（5-7 格）
- 玩家进入领地时：`alert` → `warn`（停顿 1.5 秒，显示警告符文）
- warn 期间：玩家若后退至领地外，异兽回到 `wander`；若玩家继续靠近，进入 `attack`
- 攻击后返回 `wander`，不持续追击（领地型不离开领地）
- **背刺窗口**：在 `warn` 状态时绕到异兽背后，可触发背刺压制

**🔥 主动型**
- 感知后进入 `chase`，持续追击（速度加成 × 1.3）
- 追击中周期性发动攻击（攻击间隔因异兽而异）
- 玩家使用背刺技能（见§五）可打断追击，令异兽短暂眩晕
- 距离 > 12 格才会放弃追击，进入 `wander`
- **捕获机制**：主动型异兽在"攻击硬直"后的 0.8 秒内，背后有一个有效背刺窗口

**👁 伏击型**
- 在 `wander` 状态下完全或部分隐形
- 设有"触发圈"（≤ 1.5 格）：玩家走入触发圈，异兽现身并立即攻击
- 被技能命中或攻击失败后，进入普通 `alert` 流程
- **反制方法**：兽目珠可以显示伏击型异兽位置，进入警觉状态的竹林区域内伏击型不会触发

## 四、各异兽攻击详解

### 001 玄狐（火）— 伏击型（SSR升级）

**R/SR 品质**：领地型，在草地/岩石地形巡逻
**SSR 品质**：升级为伏击型

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 火尾横扫 | 玩家进入 2.5 格正面 | 1滴 | 扇形 90°范围，0.5 秒预警动画（火焰聚集） |
| 三尾刺（SSR） | 伏击现身时 | 2滴 | 直线穿刺，玩家 1 秒内移速 -30%（火灼debuff） |

**应对策略**：正面不要进入 2.5 格；SSR 玄狐靠兽目珠或在竹林中接近（压制火感知）。

---

### 002 噬天（暗）— 主动型

噬天是游戏中**最危险**的异兽。主动追击，不轻易放弃。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 暗影撞击 | 追击距离 ≤ 2 格 | 2滴 | 正面冲撞，1 秒击退（玩家后退 1 格） |
| 光吞噬（SR） | 追击超过 10 秒 | 1滴 | AOE 黑暗：玩家视野收缩至 1.5 格，持续 5 秒 |
| 深渊吞噬（SSR） | 距离 ≤ 1 格 | 3滴 | 概率触发（30%）；触发后噬天短暂硬直 1.5 秒——唯一背刺窗口 |

**应对策略**：不建议正面对抗噬天，在已探索区域（灰暗区）接近可降低其逃跑率；使用定身技能创造背刺窗口；携带复苏符时可考虑主动引战。

---

### 003 雷翼（雷）— 主动型

雷翼不在地面追击，而是**空中俯冲**——攻击来自上方，绕后无法完全规避。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 雷击俯冲 | 追击中每 4 秒一次 | 1滴 | 闪电从雷翼位置直线落下，0.7 秒预警（闪光标记目标地格） |
| 电弧连击（SR） | 命中玩家后 | +1滴 | 连续弹射电弧，波及附近 1 格 |
| 雷暴核心（SSR） | 追击超过 8 秒 | 2滴 | 雷翼停在高空，在玩家头顶持续落雷 3 秒（1滴/秒），之后俯冲至地面——此时是背刺窗口 |

**应对策略**：雷击有地格预警，提前移开即可规避；主动引诱雷暴核心（等它降落），利用降落硬直完成背刺。

---

### 004 白泽（光）— 被动型

白泽**永不攻击**。但有独特的"凝视"机制（见 v2.0 §六），加入战斗系统后：
- 凝视期间若玩家被其他异兽攻击，白泽会短暂发出**庇护光晕**：周围 2 格内的异兽攻击力降低 50%，持续 3 秒
- 白泽的庇护光晕只触发一次，之后白泽会因感知到混乱而逃跑

**设计意图**：制造"保护白泽进行中的凝视仪式"的微型叙事——玩家需要在凝视期间挡住其他异兽的骚扰。

---

### 005 石灵（土）— 领地型

石灵速度最慢，但攻击范围大、伤害稳定，是**范围控制型**领地守卫。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 飞石 | warn 结束后，距离 >2 格 | 1滴 | 直线投射，有 0.4 秒蓄力，可侧移规避 |
| 震地（SR） | 距离 ≤ 1.5 格 | 2滴 | 以自身为圆心 2 格范围 AOE，预警：地面龟裂纹路扩散 |
| 石化气息（SSR） | 攻击命中后 | 0滴 | 附加 debuff：玩家移速 -50%，持续 3 秒（叠加两次则移速 -80%） |

**应对策略**：飞石只要不站在石灵正面直线上就能规避；震地通过拉开距离规避；被石化后优先跑向竹林降低感知，等 debuff 消退。

---

### 006 水蛟（水）— 领地型

水蛟的领地以水面为中心，**在水面附近极其危险**，在旱地则正常领地型。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 水流喷射 | 距离 ≤ 3 格（水面附近） | 1滴 | 直线水柱，0.5 秒预警，命中后玩家 2 秒内无法疾跑 |
| 旋涡拖拽（SR） | 水面格子内 | 1滴 | 若玩家误踏水边（≤0.5格），被拖入水中 1 格，触发持续伤害 |
| 龙吼（SSR） | 血量低于 50% 时触发 | 2滴 | 全方向水爆，范围 3 格，频率每 8 秒 |

**应对策略**：将水蛟从水面引离后再交战；旱地水蛟只有普通飞石攻击（水流喷射距离缩短至 1.5 格，伤害不变）。

---

### 007 风鸣（风）— 伏击型

风鸣是玩家最难预测的异兽——隐形游荡，触圈突袭，逃脱速度极快。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 风刃突袭 | 现身瞬间（触发圈激活） | 1滴 | 无预警，现身即伤害；但现身后有 0.5 秒硬直——唯一反击窗口 |
| 风压推退（SR） | 风刃后若玩家未撤离 | 1滴 | 持续推退玩家（3 秒），方向随机 |
| 隐遁撕裂（SSR） | 绕背后攻击 | 2滴 | 从玩家背后现身，玩家视野随机旋转 2 秒（迷向） |

**应对策略**：听声音（移动粒子有对应风声）预判位置；携带兽目珠可直接显现；进入竹林后风鸣失去隐身能力（竹叶阻风），可安全背刺。

---

### 008 土偶（土）— 领地型

土偶速度极慢，领地意识强，攻击有明显预警，是**最好应对**的战斗型异兽。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 泥掌拍击 | 距离 ≤ 1 格 | 2滴 | 1.0 秒预警（手臂抬起动画），原地大范围拍击，可后退 1.5 格规避 |
| 泥团投掷 | 距离 2-4 格 | 1滴 | 抛物线轨迹，有落点预警圆圈（0.5 秒），命中附加黏滞：移速 -30%，5 秒 |

**应对策略**：土偶是练习战斗的最佳对象——预警时间充足，速度极慢。泥掌是近战攻击只要拉开距离即可；泥团投掷看落点圆圈移开。

---

### 009 冰蚕（冰）— 被动型

冰蚕**永不攻击**，但与白泽不同，冰蚕没有特殊互动——它只是单纯的、拼命逃跑的小生物。

加入战斗系统后的调整：
- 被其他异兽攻击时，若攻击落在冰蚕附近 1 格范围，冰蚕会**短暂冻结**（0.5 秒不动）——这是背刺的意外窗口
- 冰蚕本身不会被"误伤"，冰冻只是其本能反应

---

### 010 墨鸦（火）— 主动型

墨鸦是速度最快的主动型异兽，采用**骚扰式打法**：快速飞过、攻击、拉开距离，不正面持续缠斗。

| 攻击名 | 触发条件 | 伤害 | 效果 |
|--------|---------|------|------|
| 墨爪掠过 | 追击飞过玩家附近（1.5 格） | 1滴 | 掠过即伤害，无预警，靠位移规避 |
| 墨迹喷射 | 每 5 秒一次 | 0滴 | 目标区域覆盖墨迹（玩家视野 -50%，20 秒），叠加最多 2 层 |
| 群鸦突袭（SSR） | 生命值正常时，追击 15 秒后 | 2滴 | 召唤 2 只幻影墨鸦同时突袭（幻影不可被封灵，消失后 SSR 本体背刺窗口开放 3 秒） |

**应对策略**：墨鸦墨迹是最难处理的 debuff，建议第一时间用驱散技能或等待消退；优先使用兽目珠清空墨迹干扰；主动型特性导致墨鸦难以背刺，群鸦突袭后的 3 秒窗口是最好机会。

## 4.5 灵契携带效果

### 4.5.1 系统概述

**捕获灵兽后，该灵兽的被动效果立即生效，持续到本次探索结束。** 不同品质（R/SR/SSR）效果递进——高品质包含低品质的全部效果。

设计原则：
- **主题契合**：效果与灵兽的元素、习性、百灵志描述一致
- **地形解锁**：部分灵兽解锁特殊地形通行能力（山地、水面），鼓励"先捕辅助兽，再探险深区"的策略
- **策略深度**：不同灵兽组合提供不同探索优势，丰富每局的抉择
- **同类不叠加**：相同类型的效果（如移速加成）取最高值，不叠加

### 4.5.3 地形解锁效果总结

| 地形 | 默认状态 | 解锁灵兽 | R级效果 | SR级效果 |
|------|---------|----------|---------|---------|
| 山地 | 不可通行 | **风鸣** | 可通行（移速 -40%） | 移速恢复正常 |
| 水面 | 不可通行 | **水蛟** | 可通行（移速 -50%） | 移速恢复正常 |

> **设计意图**：地形解锁是最具策略价值的携带效果。山地和水面在地图中形成天然屏障，将区域分割成多个孤立地带。先捕获风鸣/水蛟可以打开新路线，到达其他方式无法抵达的区域。这鼓励"先捕辅助型灵兽，再狩猎高价值目标"的策略节奏。

### 4.5.4 效果叠加规则

| 规则 | 说明 |
|------|------|
| 品质递进 | 高品质包含低品质全部效果（SSR = SSR + SR + R） |
| 同类取高 | 多只灵兽提供同类数值效果时取最高值（如雷翼 +10% 移速 和 风鸣 +15% 移速，取 +15%） |
| 独立生效 | 不同类型效果独立生效（移速 + 视野 + 减伤 各自独立计算） |
| 地形解锁 | 地形通行能力只需一只即可，不叠加 |
| debuff 免疫 | 多只提供不同 debuff 免疫时，全部生效 |

## 五、背刺技能系统

### 5.1 系统概述

每次出发前，玩家在 PrepareScreen 选择**一个背刺技能**携带入局。

背刺技能的使用时机：
1. **主动使用**：靠近任意异兽时，交互按钮旁出现技能按钮（不限背刺位置）
2. **背刺强化使用**：玩家处于背刺角度时使用技能，效果增强（标注为"背刺增强"）

技能有**使用次数限制**，局内不可补充（除非特定流派加成）。

### 5.2 投掷类技能（3种）

---

#### 🎴 灵符弹

**描述**：将封印符纸折成纸弹投出，命中异兽后令其短暂眩晕。

| 参数 | 数值 |
|------|------|
| 使用次数 | 3 次 |
| 射程 | 5 格直线 |
| 命中效果 | 眩晕 2 秒（异兽停止移动，停止攻击） |
| 背刺增强 | 眩晕延长至 3.5 秒，且本次 QTE 目标区额外扩大 10% |
| 对伏击型 | 命中使隐形异兽现身，现身后进入普通 alert 而非立即攻击 |

**适用场景**：通用型，对所有攻击类异兽都有效；新手推荐。

---

#### 🗡 追迹弹

**描述**：投出特制毒针，命中后异兽移速大幅下降，方便拉开距离或绕背。

| 参数 | 数值 |
|------|------|
| 使用次数 | 4 次 |
| 射程 | 6 格直线 |
| 命中效果 | 异兽移速 -70%，持续 6 秒 |
| 背刺增强 | 持续时间延长至 10 秒，追迹灰效果同时触发（加速调查） |
| 对主动型 | 主动型异兽中毒后会暂时放弃追击，转为 wander 状态 |

**适用场景**：主动追击型异兽（噬天、雷翼、墨鸦）；贪渊流派最佳搭档。

---

#### 💥 爆炎符

**描述**：投出一张火焰符纸，命中后引爆，对目标范围内所有实体造成伤害。

| 参数 | 数值 |
|------|------|
| 使用次数 | 2 次 |
| 射程 | 5 格，爆炸范围 1.5 格 |
| 命中效果 | 范围内所有异兽各扣 2 HP（注：异兽有独立 HP，HP 归零进入"虚弱"状态，降低封灵难度） |
| 背刺增强 | 爆炸范围扩大至 2.5 格；被击中异兽封灵成功率 +10% |
| 对群体 | 若爆炸范围内有多只异兽，全部受伤 |

> **异兽 HP 说明**：异兽有独立 HP（R:4, SR:6, SSR:8）。HP 归零不是死亡，而是"虚弱"——进入虚弱状态的异兽不能攻击、移速-50%，且封灵成功率+15%。1分钟后 HP 恢复，虚弱解除。

**适用场景**：高密度区域清场；与压制流 + 封印回响组合，可打出"爆弹→虚弱→必封"连招。

## 六、瘴气伤害机制

### 6.1 基本规则

| 参数 | 数值 |
|------|------|
| 伤害频率 | 1 滴 HP / 每 **4 秒** 在瘴气中 |
| 伤害触发 | 玩家所在地格为 `danger` 类型时计时开始 |
| 离开瘴气 | 计时重置，不累积 |
| `danger` 阶段加成 | 瘴气伤害频率加快：1 滴 / 每 **2.5 秒** |
| `collapse` 阶段 | 1 滴 / 每 **1.5 秒**（灾变阶段瘴气极度危险） |

> **v1.1 对比**：原来瘴气消耗灵石，玩家可以"穷着走"；现在消耗 HP，意味着反复穿越瘴气是真实风险，而不只是资源损耗。

### 6.2 瘴气与战斗的交互

- **主动型异兽追击进入瘴气**：异兽本身免疫瘴气伤害（它们是灵境原住民），玩家被追进瘴气时双重压力
- **领地型异兽**：领地型异兽不会主动追入瘴气区域（领地边界在瘴气外），给玩家"把异兽拖进瘴气"这个策略创造了可能性
- **驱散法净化**：驱散法可临时将瘴气变为草地，是高危区探索的关键工具

### 6.3 瘴气视觉反馈增强

- 玩家每受到一次瘴气伤害，屏幕边缘产生一圈浓墨渗透动画（比普通受击更缓慢、更压迫）
- 血量 ≤ 3 时，玩家角色有轻微抖动的虚弱特效（不影响操控，但视觉上传递紧张感）

典型的道具格决策：

## 十、战斗与核心循环的整合

### 10.1 一局典型战斗流程示例

```
【发现水蛟】（领地型，水面附近）

1. 玩家侦察：通过巢穴线索判断是水蛟，意识到水面区域危险
2. 路线规划：绕过水面，从草地方向接近，规避水蛟速度加成
3. 进入领地（5格内）→ 水蛟进入 alert
4. warn 状态触发（1.5秒警告）
   ↓
5. 选择A：继续强冲 → 水蛟进入 attack → 水流喷射（1滴）→ 正面QTE（无背刺加成）
   选择B：使用灵符弹命中 → 水蛟眩晕2秒 → 趁眩晕绕背 → 背刺状态 + 背刺增强（眩晕3.5秒）→ QTE
   选择C：等warn消失，绕到水蛟背后（领地型在warn期间不会转身）→ 无技能背刺 → 标准QTE

6. QTE成功 → 手选封灵器（考虑背包剩余和血量）→ 封灵判定
7. 成功 → 捕获动画 + 属性被动（水蛟捕获不增加收益，但避免水面区域威胁）
```

### 10.2 战斗类型对探索策略的影响

| 场景 | 推荐应对 |
|------|---------|
| 噬天（主动）在高危区 | 先用定身咒冻结 → 快速背刺 → 若失败立即用疾风符脱战 |
| 瘴气区有石灵（领地） | 驱散法清出安全通道 → 绕后 → 在干净地形上QTE |
| 风鸣（伏击）密度高的区域 | 使用兽目珠 + 提前进竹林 → 所有风鸣失去隐身 → 安全接近 |
| 血量 ≤ 3 时发现 SSR | 权衡：消耗绛珠露回血再打，还是直接用高品级封灵器快速结束 |
| collapse 阶段撤离途中遇敌 | 优先使用疾风符加速脱战，不建议在 collapse 阶段新开战斗 |

### 10.3 死亡对捕获率的设计影响

引入战斗后，以下情况会让玩家"理性放弃"某只异兽：

- 血量 ≤ 2，前面有主动型异兽，且无恢复道具
- collapse 阶段，异兽与撤离点在反方向
- 封灵器已耗尽，仅剩T1，且目标是SSR

这些放弃的时刻比"一定要捕"更能制造玩家与异兽之间的情感张力。
