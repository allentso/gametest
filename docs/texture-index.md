## 一、当前状态概述

已完成 P1（资源 6 张 + 线索 3 张）和 P2（撤离点 1 张）共 **10 张 128×128 透明 PNG**。新增 3 项待制作贴图（气息印记线索 + 兽目珠/封印回响资源）。

**渲染架构**：贴图优先 + 矢量 fallback 双保险
- `InkRenderer.initImages(nvg)` 在 `Start()` 中加载所有贴图
- 每个绘制函数先检查贴图 handle，有则用贴图渲染，无则自动回退到矢量绘制
- 矢量 fallback 可作为贴图加载失败时的降级方案，保证任何情况下都有可用视觉

**贴图加载入口**：`scripts/render/InkRenderer.lua` → `InkRenderer.initImages(nvg)`
**贴图存放目录**：`assets/image/`

## 七、贴图制作规范

新增贴图时遵循以下规范：

1. **格式**：PNG，透明背景（alpha 通道）
2. **尺寸**：统一 128×128px（地图实体图标）
3. **风格**：水墨白描 / 国风手绘 / 宣纸质感，颜色参考 `InkPalette.lua`
4. **命名规则**：`<类型>_<名称>_<日期时间戳>.png`
5. **存放路径**：`assets/image/`
6. **代码注册**：在 `InkRenderer.lua` 的 `IMAGE_PATHS` 表中添加 key-path 映射
7. **加载方式**：`InkRenderer.initImages(nvg)` 在 `Start()` 中自动加载所有注册贴图
