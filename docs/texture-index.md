**贴图加载入口**：`scripts/render/InkRenderer.lua` → `InkRenderer.initImages(nvg)`
**贴图存放目录**：`assets/image/`

## 七、贴图制作规范

如需替换矢量 fallback 为正式贴图：

1. **格式**：PNG，透明背景（alpha 通道）
2. **尺寸**：统一 64×64px（地图实体图标），可 @2x 提供 128×128 高清版
3. **风格**：水墨白描 / 国风手绘 / 宣纸质感，颜色参考 `InkPalette.lua`
4. **命名规则**：`<类型>_<名称>_<日期时间戳>.png`（已有命名无需改动）
5. **存放路径**：`gametest/assets/image/`
6. **加载方式**：放入 `assets/image/` 目录后，`InkRenderer.initImages` 会自动加载

### 替换优先级建议

| 优先级 | 贴图 | 理由 |
|-------|------|------|
| P1 | 资源图标（R-01 ~ R-06） | 游戏核心收集物，需要第一眼辨认 |
| P1 | 线索图标（C-01 ~ C-03） | 核心玩法入口，高频交互 |
| P2 | 撤离点（E-01） | 关键决策点，但出现频率低 |
| P3 | 地形纹理 | 程序化已够用，替换收益低 |
| P3 | 异兽贴图 | 程序化白描已有特色，可留后期精修 |
