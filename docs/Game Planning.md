# 《山海异闻录：寻光》游戏策划文档

> **定位**: 山海经水墨国风 · 高风险捉宠撤离 · 闪光收集驱动
> **平台**: 手机竖屏
> **单局时长**: 6-8 分钟
> **视觉方向**: 国风水墨（详见 ui.md）
> **技术方案**: 详见 technical planning.md

---

## 一、产品定位

### 1.1 一句话描述

> 进入灵境追踪异兽踪迹，完成封印捕获，在灾变来临前撤离——一款山海经风格的高风险捉宠撤离游戏。

### 1.2 核心循环

三段式循环，服务同一目标：**把稀有异兽活着带出灵境，点亮图鉴。**

- **搜**：探索迷雾笼罩的灵境，采集资源，追踪线索，锁定异兽
- **压制**：控场、削韧、封印的短时高压 QTE 周旋
- **撤**：带着未稳固灵契安全撤离——"捕获成功"不等于"真正拿到"

### 1.3 核心卖点

1. **猎踪封灵**——稀有异兽不是"抽"出来的，是"猎"出来的
2. **贪婪博弈**——已经抓到异色了，要不要继续赌闪光？
3. **水墨山海**——宣纸为卷、墨色为骨、朱砂点睛

### 1.4 目标用户

| 类型 | 占比 | 特征 |
|------|------|------|
| 主力用户 | 70% | 16-30岁，喜欢概率刺激且希望操作影响结果，碎片化场景 |
| 次级用户 | 20% | roguelike/搜打撤玩家，被轻量化+收集驱动吸引 |
| 长尾用户 | 10% | 山海经/国风爱好者，强迫症收集玩家 |

## 二、单局玩法循环

### 2.1 六段式循环

```
① 进场准备 → ② 搜索与追踪 → ③ 发现异兽 → ④ 封印捕获
      ↑                                    │
      │              ⑤ 撤离决策 ←──────────┘
      │                   │
      └─── ⑥ 结算入库 ←──┘
```

### 2.2 第一段：进场准备（局前，15-30秒）

选择携带装备进入灵境：

| 槽位 | 内容 | 决策维度 |
|------|------|---------|
| 封灵器 | 选择类型与数量（上限6个） | 赌大的带天命盘？还是稳妥多带素灵符？ |
| 探测道具 | 寻迹盘/妖气罗盘/显形符 | 影响发现稀有异兽的效率 |
| 生存物资 | 回复丹/隐匿符/疾行靴 | 影响存活和撤离成功率 |

### 2.3 第二段：搜索与追踪（0:00-4:00）

进入灵境后，地图被**战争迷雾**覆盖，玩家只能看到周围 4-5 格视野范围。

> 设计原则：进图 20 秒内遇到第一个可交互对象，60 秒内遇到第一只 R 异兽。

**三类资源**：

| 类型 | 举例 | 获取 | 用途 |
|------|------|------|------|
| 基础资源 | 灵石、草药 | 地图随处可见 | 日常成长，撤离保留 |
| 稀有资源 | 兽魂、天晶 | 危险区域/事件奖励 | 高级合成，撤离失败丢失 |
| 本局资源 | 追迹灰、镇灵砂、归魂符 | 矿石伴生/遗迹/高危点 | 仅本局有效，直接服务捕获流程 |

**追迹灰获取**（独立于线索调查，避免循环依赖）：
- 采集灵石矿时 30% 概率伴生
- 随机事件奖励
- 新区域首次探索奖励

**异兽发现：发现方式决定品质**

不存在遭遇后的品质 roll。玩家搜索行为直接决定遇到什么品质：

| 品质 | 触发条件 | 频率 |
|------|---------|------|
| R 普通 | 地图自然游荡 | 每 40-80 秒刷新一只 |
| SR 异色 | 收集 3 个线索 | 每局 1-3 次机会 |
| SSR 闪光 | 收集 5 个线索 + 通过闪光判定 | 平均 3-5 局触发 1 次 |

**追踪系统**：
- 地图散布"足迹"、"元素残响"、"巢穴波动"三种线索
- 调查时间：无追迹灰 2 秒，有追迹灰 0.5 秒
- 3 线索 → SR 异色异兽出现
- 5 线索 + 闪光判定 → SSR 闪光异兽出现（未通过则出 SR）

### 2.4 第三段：异兽朝向与偷袭

每只异兽有朝向方向（基于移动方向）。玩家的接触角度决定战术优势：

| 接触方向 | 角度范围 | 效果 |
|---------|---------|------|
| 背后偷袭 | 异兽背面 ±60° | 捕获率 +20%，压制窗口 +30% |
| 侧面接触 | ±60°~120° | 无额外效果 |
| 正面接触 | ±120°~180° | 50% 概率异兽直接逃跑 |

SSR 异兽感知范围更大（6格），背后偷袭几乎是捕获 SSR 的**必要策略**。

### 2.5 第四段：封印捕获（单次 3-6 秒）

接触异兽后进入短时高压封印事件：

```
0.0s  接触异兽 → 画面墨色暗化聚焦
0.5s  压制阶段 → QTE 操作
1.5s  压制成功 → 自动选择最高级封灵器 → 投掷
2.0s  封灵器摇晃（R:1次 / SR:2次 / SSR:3次）
3.5s  封印判定 → 成功/失败
4.0s  成功 → 生成【未稳固灵契】（需撤离确认）
      失败 → 异兽转为警觉态（移速加快，再次接近更难）
```

**压制操作**（两种模式）：

| 模式 | 操作 | 适用 |
|------|------|------|
| 时机点击 | 指针摆动经过目标区域时点击 | R（1次宽窗口）/ SR（2次窄窗口） |
| 连续封印 | 3 秒内快速点击达到 8 次 | SSR |

### 2.6 第五段：撤离决策

**搜灵撤核心快感：已经赚了，要不要继续贪？**

- 地图上有**固定撤离点**（地图边缘，需探索发现）
- **灾变倒计时** 8 分钟——超时强制结算，未稳固灵契全部丢失
- 玩家必须**控制角色走到撤离点**，站定 3 秒完成撤离（无 UI 撤离按钮）

**四种博弈场景**：

| 场景 | 已拥有 | 诱惑 | 风险 |
|------|-------|------|------|
| A | 2只R异兽 | 附近有异色线索 | 时间过半 |
| B | 1只SR异兽 | 差1条线索触发闪光 | 仅剩2分钟 |
| C | 1只SSR闪光！ | 还有空间+天晶矿 | 撤离点在远端 |
| D | 满背包 | 无 | 灾变倒计时30秒 |

### 2.7 第六段：结算入库

| 撤离状态 | 基础资源 | 稀有资源 | 未稳固灵契 |
|---------|---------|---------|-----------|
| 成功撤离 | 全部保留 | 全部保留 | **正式入库，点亮图鉴** |
| 灾变超时 | 保留50% | 全部丢失 | **全部丢失** |

> 无"中途倒下"机制——不设生命值/死亡，撤离失败唯一原因是灾变超时。

**见闻机制**：即使撤离失败，图鉴记录"曾经见过"（灰色剪影），激发复玩动力。

### 2.8 节奏时间轴

```
紧张度
  ↑
  │                              ★闪光捕获!?
  │                             / \
  │         SR追踪成功!       /   \    灾变
  │            /  \          /     \  逼近!
  │  首只R!  /    \   压制  /       \  / \
  │   / \   / 足迹  \  / \    撤离  \/   │
  │  / 灵石\/  线索   \/   \  博弈       │
  │ / 采集                   \          结算
  │/  足迹                    \
  └──────────────────────────────────────→ 时间
  0    1    2    3    4    5    6    7  8 分钟
  
  ← 搜索与追踪 → ← 捕获高压 → ← 撤离博弈 →
```

| 等级 | 名称 | 合成成本 | 捕获率 | 特殊效果 |

- 未使用的封灵器撤离失败也保留

## 六、概率与保底系统

### 6.1 闪光判定（唯一保留概率的环节）

收集满 5 线索后进行闪光判定：

| 条件 | 效果 |
|------|------|
| 基础通过率 | 15% |
| 每多 1 线索（超过5个） | +5% |
| 天命盘加成 | +15% |
| 判定未通过 | 出现 SR（保底不空手） |

### 6.2 SSR 保底（跨局累计，永不重置）

以"捕获成功次数"为单位：

| 累计次数（未出SSR） | 闪光判定加成 |
|-------------------|------------|
| 0-19 | 无 |
| 20-39 | +10% |
| 40-59 | +20% |
| 60-79 | +35% |
| 80 | **硬保底：必定闪光** |
| 出SSR后 | 归零 |

### 6.3 SR 保底（跨局累计）

| 累计次数（未出SR/SSR） | 效果 |
|----------------------|------|
| 0-9 | 无 |
| 10-14 | 3线索即触发SR |
| 15 | **硬保底：下一只强制SR** |
| 出SR或SSR后 | 归零 |

## 七、灵境地图

### 7.1 地图结构

竖屏纵深地图（20×30 格），战争迷雾覆盖：

- **出生点**在底部中央安全区
- 越往上（深入）越危险
- 玩家视野半径 4-5 格，其余为墨色迷雾
- 走过的区域变为"已探索"（灰色记忆），但不再显示异兽/资源
- 撤离点在地图边缘，需探索发现

区域划分（从外向内）：

| 区域 | 分布 | 内容 |
|------|------|------|
| 安全区 | 地图边缘2格 | 出生点、撤离点 |
| 搜索区 | 3-5格深 | 基础资源、R异兽、低级线索 |
| 稀有区 | 6-9格深 | 稀有资源、高级线索、SR触发 |
| 高危区 | 10+格深 | 天晶、归魂符、SSR触发点 |

### 7.2 灾变机制

| 阶段 | 时间 | 地图变化 | 提示 |
|------|------|---------|------|
| 平静期 | 0:00-4:00 | 正常 | 无 |
| 预警期 | 4:00-5:30 | 边缘开始瘴气 | "灵境开始不稳定..." |
| 危险期 | 5:30-7:00 | 瘴气扩散 | "灾变即将来临！尽快撤离！" |
| 灾变期 | 7:00-8:00 | 全图瘴气 | "灵境崩溃中！" |
| 崩溃 | 8:00 | 强制结算 | 未撤离→稀有资源和灵契全部丢失 |

灾变瘴气从地图边缘向内收缩，**吞噬已探索区域变回黑暗**，形成"墨色洪水"的视觉压迫。

## 八、撤离与灵契稳定

### 8.1 撤离方式

| 类型 | 位置 | 耗时 |
|------|------|------|
| 固定撤离点 | 地图边缘安全区 | 站定3秒 |

> 撤离是**物理行为**——玩家必须控制角色走到撤离点。无 UI 撤离按钮。

### 8.2 灵契稳定性（可操作事件）

撤离时灵契可能发生不稳定事件——有预兆、可操作，**不是静默骰子**：

| 灵契品质 | 触发概率 | 归魂符效果 |
|---------|---------|-----------|
| R | 5% | 完全抑制 |
| SR | 30% | 降至10% |
| SSR | 50% | 降至20% |

事件流程：灵契闪烁预警(1秒) → QTE时机点击 → 成功=稳固 / 失败=灵契破碎丢失

## 九、图鉴系统

### 9.1 三种状态

| 状态 | 触发条件 | 显示 | 驱动力 |
|------|---------|------|-------|
| 未发现 | 从未遭遇 | 灰色剪影 + "???" | "还有什么异兽？" |
| 见闻 | 遭遇过但未带出 | 半透明+名称 | "我见过但没带出来！" |
| 已收录 | 成功撤离带出 | 完整展示+品质标记 | "还差哪个品质？" |

### 9.2 图鉴奖励

| 完成度 | 奖励 |
|-------|------|
| 收集5只不同异兽 | 妖气罗盘配方 + 5兽魂 |
| 收集8只 | 解锁雷峰灵境 |
| 集齐某异兽全3品质 | 专属头像框 |
