# 《山海异闻录：寻光》视觉与 UI 渲染规格

> **渲染技术**: NanoVG 矢量绘图（全屏纯代码渲染，无位图资源）
> **屏幕方向**: 竖屏 Portrait
> **分辨率模式**: 模式 B（`logW = physW/dpr, logH = physH/dpr`）

---

## 一、核心视觉约束（必须遵守）

### 1.1 绝对禁止事项

以下事项会导致画面偏离水墨风格，**任何屏幕、任何渲染函数中都不允许**：

| 禁止事项 | 说明 | 正确做法 |
|---------|------|---------|
| ❌ 画格线/网格线 | 地图是连续水墨画面，不是消除游戏 | 瓦片之间靠自然衰减过渡，无任何边框 |
| ❌ 硬边矩形填色做瓦片 | `nvgRect` + 纯色填充 = 彩色方块 | 用 `nvgRadialGradient` 从中心向外衰减至透明 |
| ❌ 图标化实体 | 圆圈+文字符号 = 超市价签 | 用水墨笔触组合成具象化的白描形态 |
| ❌ 高饱和度大面积色块 | 破坏水墨淡雅感 | 所有颜色 alpha ≤ 0.45，靠层叠建立浓度 |
| ❌ 现代扁平UI控件 | 破坏国风意境 | 按钮用墨色描边+宣纸底色+毛笔笔触边缘 |
| ❌ 交互型品质选择器 | 品质由玩家行为决定，非手动选择 | 品质印章为**只读条件指示**，非控件 |
| ❌ 撤离UI按钮 | 撤离是物理行为（走到撤离点） | 撤离点是地图上的实体，不是UI按钮 |
| ❌ 局内显示跨局资源 | 灵石/兽魂/天晶是局间资源 | 局内只显示本局道具（追迹灰/镇灵砂/归魂符） |

### 1.2 核心视觉原则

1. **宣纸为底**：所有屏幕全局背景色 = `paper(0.96, 0.93, 0.87)`
2. **墨色为骨**：信息层级靠墨色五阶（焦→浓→重→淡→清）区分，不靠颜色区分
3. **留白即呼吸**：信息区域之间保持大面积宣纸底色留白，不用分割线填满
4. **朱砂点睛**：整个屏幕中朱砂红只出现在 1-2 个最关键的元素上（如倒计时、灵契印章）
5. **连续画面**：地图是一幅展开的水墨画卷，不是方格拼图
6. **笔触有生命**：所有线条通过 `BrushStrokes.inkLine` 绘制，有粗细变化和飞白断笔

## 二、色彩体系 — InkPalette

### 4.4 中部：游戏地图区域

#### 4.4.1 地图渲染基本规则

**最关键约束**：地图是一幅连续的水墨画卷，不是方格拼图。

```
正确 ✅                          错误 ❌
┌─────────────────┐             ┌──┬──┬──┬──┬──┐
│                  │             │绿│绿│棕│蓝│绿│
│   ～ 淡墨晕 ～    │             ├──┼──┼──┼──┼──┤
│  ≋ 水墨水纹 ≋    │             │棕│灰│绿│绿│棕│
│       ⊿ 皴法岩   │             ├──┼──┼──┼──┼──┤
│    ⌇ 兰草摇曳    │             │蓝│绿│棕│灰│绿│
│                  │             └──┴──┴──┴──┴──┘
└─────────────────┘             ← 消除游戏，禁止
← 水墨画卷，正确
```

#### 4.4.2 瓦片渲染（InkTileRenderer）

瓦片是逻辑单位（碰撞/寻路用），**视觉上不存在边界**：

| 地形 | 渲染方式 | alpha 范围 |
|------|---------|-----------|
| grass（草地） | 宣纸底色上画 2-3 笔兰草弧线（`nvgQuadTo`），随风轻摆 | 草笔触 0.15-0.30 |
| rock（岩石） | `BrushStrokes.cunTexture` 皴法纹理（6-8笔短促线条） | 0.15-0.25 |
| water（水面） | 2条柔和贝塞尔水纹（`nvgBezierTo`），随时间缓流 | 0.15-0.25 |
| path（小径） | 极淡的赭石色径向晕染，几乎融入宣纸 | 0.08-0.15 |
| bamboo（竹林） | 2根竖向墨线 + 随风轻摆 | 主干 0.4-0.5 |
| danger（瘴气） | 暗朱色径向晕染脉冲 | 0.10-0.20 |

**每种瓦片的渲染核心**：
1. 先用 `nvgRadialGradient` 从瓦片中心向外画底色晕染（alpha ≤ 0.35），**渐变到完全透明**
2. 再叠加笔触细节（草笔/皴法/水纹等）
3. 相邻瓦片的晕染自然重叠，形成**连续的有机画面**
4. **绝不画任何矩形边框**

#### 4.4.3 战争迷雾渲染

| 迷雾状态 | 视觉效果 |
|---------|---------|
| DARK（未探索） | 不绘制任何瓦片或实体，该区域保持宣纸底色上叠加的纯墨色遮罩 |
| EXPLORED（已探索） | 瓦片正常绘制但所有 alpha 降至 30%，叠加淡墨径向晕染（不是半透明矩形色块！），不显示异兽/资源/线索 |
| VISIBLE（当前视野） | 正常渲染 |

**迷雾边缘**：从可见区域到黑暗区域之间使用**径向渐变柔化过渡**（`nvgRadialGradient`），不是硬边切割。以玩家屏幕坐标为中心，视野半径×PPU为内径，外径比内径大 15-20%，从 alpha 0 渐变到 alpha 0.9。

#### 4.4.4 实体渲染——具象化水墨，非图标

**玩家角色**：

| 元素 | 绘制方式 |
|------|---------|
| 主体 | 从上方俯视的水墨斗笠：浓墨实心圆 (r×0.55) + 淡墨描边圆 (r×0.7) + 顶部小墨点 (r×0.08) |
| 呼吸 | 描边圆半径 sin(t×2)×0.04 脉动 |
| 方向指示 | 移动方向上画一笔朱砂短线 (r×0.9 长, 2px 宽, alpha 0.5) |
| 行迹 | 移动时身后生成 2-3 个递减 alpha 的淡墨 inkDot（模拟脚步墨尘，0.3s 后消失） |
| 阴影 | 脚下 inkWash 椭圆晕染 (r×0.2 内径, r×0.5 外径, alpha 0.20) |

**线索**（三种类型）：

| 类型 | 绘制方式 |
|------|---------|
| 足迹 | 地面上 3 个递减大小的淡墨爪印（`inkDot` 椭圆变体），间距 r×0.4，alpha 0.3-0.5 |
| 元素残响 | 3 个同心圆描边脉冲（azure 色, alpha 0.10-0.20），由内向外扩散 |
| 巢穴 | 皴法纹理堆 (4笔) + 中心一个浓墨点 (alpha 0.7)，周围碎石墨点 |

**资源点**（具象化白描，不是圆圈+文字）：

| 资源 | 绘制方式 |
|------|---------|
| 灵石矿 | 3-4笔 `cunTexture` 皴法组成嶙峋岩块 + 顶部极淡翡翠色 `inkWash` 光晕 (alpha 0.08-0.12)。接近时光晕变亮 (→0.25) |
| 天晶 | 菱形墨线轮廓 + 内部 gold 色晕染 (alpha 0.12) + 缓慢旋转 |
| 追迹灰 | 地面上的散落粉末——5-7个极淡 inkDot 随机分布 (alpha 0.15-0.25) |

**撤离点**：

| 元素 | 绘制方式 |
|------|---------|
| 标记 | 3 层同心 jade 色描边环 (alpha 0.35-0.12)，缓慢呼吸脉动 |
| 中心 | gold 色小墨点 (alpha 0.5 + sin(t×2)×0.2) |
| 撤离中 | 外层变为进度弧 (`nvgArc` gold 色, 3px, 随进度填充) |

**异兽**：

| 元素 | 绘制方式 |
|------|---------|
| 主体 | 由 `BeastRenderer.shapes[beast.type]` 定义的水墨画法形态（详见策划案异兽外观方向） |
| 品质光晕 | R=无, SR=1层 azure 色墨晕环, SSR=3层 gold 色墨晕环+金色墨迹粒子旋转 |
| 朝向指示 | 异兽前方淡墨扇形（±60° 视野锥, alpha 0.05-0.11），让玩家判断偷袭角度 |
| 警觉状态 | 头顶朱砂色 "!" 字（弹跳动画） |
| 逃跑状态 | 身后 3 道速度线（递减 alpha 的短墨线） |

## 五、SuppressOverlay — 压制 QTE

**模态覆盖层**，推入时底层暂停。

```
┌──────────────────────┐
│  (半透明墨色遮罩 70%)   │
│                      │
│   【SR·石灵】 22px    │  ← 品质色异兽名
│                      │
│  ═══╠══█══╣═══════  │  ← 时机模式
│     ↑     ↑          │
│   目标区  指针         │
│                      │
│  封印 1/2             │
│                      │
│  ── 或（连续模式）──   │
│      ╭───╮           │  ← 环形墨环
│      │ 8 │           │    中央计数
│      ╰───╯           │
│  ═══════════════     │  ← 倒计时条
└──────────────────────┘
```

### 时机模式

| 元素 | 规格 |
|------|------|
| 横条 | 宽 logW×0.6, 高 18px, 圆角 4px, inkWash 50% 底 |
| 目标区 | jade 40% 填充 + sin(t×4)×0.1 脉冲 + 70% 描边 |
| 指针 | 3px inkDark 竖线 + 顶端 5px 墨点 |
| 命中计数 | 目标区下方 +16px, 13px inkMedium |

### 连续模式

| 元素 | 规格 |
|------|------|
| 底环 | 外径55px/内径40px, inkWash 30% |
| 进度弧 | -90°顺时针, gold 60% |
| 中央计数 | 28px inkDark |
| 倒计时条 | 宽 logW×0.4, 高6px, jade→cinnabar 渐变 |

### 反馈

- 命中：120px gold 色圆形闪光 30%→0 衰减
- 震屏：0.15s 高频正弦偏移（4px）
- 结果：36px 文字 + 墨晕光环，1.2s 后自动 pop

## 六、CaptureOverlay — 捕获演出

**全屏模态，4.5 秒电影级揭示动画。**

```
时间轴: 0s ── 1.0s ── 2.0s ── 3.2s ── 4.5s
        │      │       │       │       │
     墨染扩散 金光乍现  符文浮现 品质揭晓 自动关闭
```

## 十二、视觉哲学总结

> **水墨留白 · 以简驭繁**

1. 宣纸底色统一——所有屏幕同一暖白底
2. 墨色五阶层次——信息层级靠浓淡区分
3. 点缀色克制——一屏最多2种点缀色高亮

5. 连续画面——地图是画卷不是格子
6. 印章意象——灵契/品质标签用方印风格
7. 纯矢量无位图——零图片依赖
